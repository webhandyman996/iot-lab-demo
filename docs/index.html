物聯網實習教學範例（IoT Lab Demo）

這些是 Arduino / ESP8266 / ESP32 硬體程式，需要實際開發板與感測器才能執行，本頁提供程式說明與原始碼連結，供學生對照課堂使用的硬體閱讀與練習。

開發板連網範例（Board-to-Web）

Arduino Mega2560 讀取感測器資料，透過序列埠指令控制旁邊的 ESP32 連上 WiFi，再把資料以 HTTP POST 送到後端 API。

硬體：Arduino Mega2560 + 感測器；WiFi帳密與API金鑰請自行填入 config.h
